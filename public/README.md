# Public Assets Folder

This folder contains static assets that are served directly by Vite.

## FD-258 Fingerprint Card

To include a local copy of the FD-258 fingerprint card PDF:

1. Download the official FD-258 form from the FBI website
2. Save it as `fd-258.pdf` in this folder
3. The download button on the DIY Guide page will automatically use this file if it exists

**Note:** The FD-258 form is a government document. Always verify you have the most current version from the official FBI website.
